Vorbereitung

Um interne Links im Wörterbuch beizubehalten, wird empfohlen, unter System settings > Edit folgende Einstellungen vorzunehmen:

Enable SEF (Search engine friendly URLs)

"Link creation mode": Text (&id=Term)

Installation

Verwende folgende Informationen, um in Glossword zunächst ein leeres Wörterbuch zu erstellen: Dictionaries > Add.

"Title":

Deutsch-Tigrinisches Wörterbuch

"Short description":

Deutsch-Tigrinisches Wörterbuch

"Description":

Copy and paste the raw HTML code to the "description" text area in the dictionary settings.

Vorschau

Deutsch-Tigrinisches Wörterbuch

Quellen

Das Wörterbuch basiert auf folgenden Glossaren:

  • Wörterbuch von Binijam Tesfay und Gertrud Danieli
  • Tigrinja Wörterbuch, enthält insbesondere auch Wörter aus dem religiösen Bereich
  • Tigrinya Translate Phrasebook, ausgelegt für die Flüchtlingsarbeit

Die Transliteration wurde vom Autor gemäß des Wikipedia-Artikels für alle Einträge vereinheitlicht.

Autor: Martin Podolak
Version: 1.0 (20.02.2019)

"Dictionary elements settings":

Definition

In allen anderen elements settings den Haken herausnehmen.

Import der Wörterbucheinträge

Importiere die Wörterbucheinträge danach ins neue Wörterbuch mithilfe der XML-Dateien aus diesem Verzeichnis: Terms > Import > Auswahl des neuen Wörterbuchs.
Installiere hiernach einzeln die XML-Dateien.
